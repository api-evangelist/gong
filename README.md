## APIs
- **Gong Calls API** — List calls, retrieve metadata, transcripts, recordings and trackers.
- **Gong Users API** — Read user, manager and team data.
- **Gong CRM Integration API** — Push CRM-style account, contact, deal and engagement data into Gong.
- **Gong Engagement Data API** — Read activity, scorecards, trackers, and library content.
- **Gong Library API** — Access and organize library folders and shared call collections.
