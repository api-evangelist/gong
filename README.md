# Gong (gong)

Gong is a revenue intelligence platform that captures and analyzes customer interactions across phone, web conferencing, and email to provide insights for sales teams.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/gong/refs/heads/main/apis.yml)

## APIs

### Gong Calls API
Endpoints to retrieve, add, and manage call data including recordings, transcripts, and media files. Supports filtering calls by date range, retrieving detailed call analytics, and uploading call recordings from external telephony systems.

- **Documentation:** https://help.gong.io/docs/what-the-gong-api-provides
- **OpenAPI:** [openapi/gong-calls-openapi.yml](openapi/gong-calls-openapi.yml)

### Gong Users API
Retrieve user information, user histories, and filter users by various criteria. Provides access to team member profiles and licensing details.

- **OpenAPI:** [openapi/gong-users-openapi.yml](openapi/gong-users-openapi.yml)

### Gong Stats API
Activity metrics including user interaction statistics, answered scorecards, talk ratios, and aggregated user activity for performance tracking and analytics.

- **OpenAPI:** [openapi/gong-stats-openapi.yml](openapi/gong-stats-openapi.yml)

### Gong CRM API
Integrate with CRM systems by registering integrations, uploading CRM objects and schemas, mapping users and fields, and managing associations between calls and CRM records.

- **OpenAPI:** [openapi/gong-crm-openapi.yml](openapi/gong-crm-openapi.yml)

### Gong Engage API
Manage and customize Engage flows at scale, list available flows and folders, assign or unassign prospects, and override flow content for tailored campaigns.

- **OpenAPI:** [openapi/gong-engage-openapi.yml](openapi/gong-engage-openapi.yml)

### Gong Settings API
Retrieve scorecard configurations and list company workspaces for programmatic access to organizational settings and evaluation frameworks.

- **OpenAPI:** [openapi/gong-settings-openapi.yml](openapi/gong-settings-openapi.yml)

### Gong Library API
Access library structure and calls stored in specific folders for programmatic browsing and retrieval of curated call content.

- **OpenAPI:** [openapi/gong-library-openapi.yml](openapi/gong-library-openapi.yml)

### Gong Permissions API
Manage access controls through permission profiles and user call access assignments.

- **OpenAPI:** [openapi/gong-permissions-openapi.yml](openapi/gong-permissions-openapi.yml)

### Gong Data Privacy API
Find references to email addresses and phone numbers, and purge associated data to support compliance with privacy regulations.

- **OpenAPI:** [openapi/gong-data-privacy-openapi.yml](openapi/gong-data-privacy-openapi.yml)

### Gong Auditing API
Retrieve audit log data by type and time range for compliance and security monitoring.

- **OpenAPI:** [openapi/gong-auditing-openapi.yml](openapi/gong-auditing-openapi.yml)

### Gong Meetings API
Create, update, and delete Gong meetings, and validate meeting integration status.

- **OpenAPI:** [openapi/gong-meetings-openapi.yml](openapi/gong-meetings-openapi.yml)

### Gong Engagement API
Report customer engagement events including content sharing, content viewing, and custom actions.

- **OpenAPI:** [openapi/gong-engagement-openapi.yml](openapi/gong-engagement-openapi.yml)
